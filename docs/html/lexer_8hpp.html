Produces a lexer with a list of tokens from the source code of filepath. Any prevalent keywords should be provided in keywords. The identifier(s) for a SINGLE LINE comment is provided as comment.

Parameters
filepathThe filepath to read the source code from
keywordsA vector of strings that are the keywords of the language
typesA vector of strings that specify the types in the language
commentWhat a single line comment is in the language
