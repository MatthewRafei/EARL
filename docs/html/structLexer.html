Detailed Description

The API for lexical analysis of a document.

A Lexer (for lexical analysis) https://en.wikipedia.org/wiki/Lexical_analysis is a tool that splits up and catagorizes individual 'tokens' for parsers. This implementation constructs a linked list of tokens for the parsers to traverse. It starts at m_hd and ends at m_tl.
