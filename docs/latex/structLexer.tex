\doxysubsection{Detailed Description}
The API for lexical analysis of a document. 

A \doxylink{structLexer}{Lexer} (for lexical analysis) \href{https://en.wikipedia.org/wiki/Lexical_analysis}{\texttt{ https\+://en.\+wikipedia.\+org/wiki/\+Lexical\+\_\+analysis}} is a tool that splits up and catagorizes individual \textquotesingle{}tokens\textquotesingle{} for parsers. This implementation constructs a linked list of tokens for the parsers to traverse. It starts at {\ttfamily m\+\_\+hd} and ends at {\ttfamily m\+\_\+tl}.
